# Human-in-the-Loop Sales: When AI Should Hand Off to Founders

AI agents handle 95% of sales activities autonomously. The other 5% requires human judgment, creativity, and relationship-building.

Knowing which 5% is the difference between scaling efficiently and losing critical deals.

## The 95/5 Rule

In 2026, optimal sales operations follow the 95/5 split:

**95% Autonomous (AI Handles):**
- Prospecting and lead sourcing
- Initial outreach and follow-ups
- Simple qualification questions
- Meeting scheduling
- Content delivery
- Basic objection handling
- Routine check-ins
- Data entry and CRM updates

**5% Human (Founder Handles):**
- High-value deal negotiations
- Complex custom requirements
- Executive relationship building
- Strategic partnerships
- Crisis management
- Final pricing approvals
- Contract negotiations
- Vision-level conversations

        pauseUntilResolved: true
      }
    },

    {
      trigger: 'enterprise-custom-pricing',

        request.seats > 100 ||
        request.includes('enterprise agreement')
      ),
      action: {
        notify: 'slack:founders',
        priority: 'high',
        include: ['deal-size-estimate', 'company-profile', 'conversation-history']
      }
    },

    {
      trigger: 'acquisition-partnership-inquiry',
      detect: ({ message }) => containsKeywords(message, [
        'acquire', 'acquisition', 'partnership', 'strategic alliance',
        'investment opportunity', 'merger'
      ]),
      action: {
        notify: 'slack:founders',
        priority: 'urgent',
        response: 'human-will-respond-within-24h'
      }
    },

    {
      trigger: 'negative-sentiment-escalation',
      detect: ({ message, sentiment }) => (
        sentiment.score < -0.7 ||
        sentiment.emotion === 'angry' ||
        message.includes('cancel') && sentiment.frustrated
      ),
      action: {
        notify: 'slack:founders',
        priority: 'high',
        draftApology: true,
        suggestRecoveryPath: true
      }
    }
  ],

  // Escalate if conditions met
  conditional: [
    {
      name: 'high-value-opportunity',
      detect: async ({ lead }) => {
        const dealSize = await estimateDealSize(lead)
        return dealSize > 50000 // $50K+ ARR
      },
      action: {
        notify: 'slack:sales',
        priority: 'medium',
        assignHuman: 'senior-ae',
        aiContinuesUntil: 'demo-scheduled'
      }
    },

    {
      name: 'complex-technical-question',
      detect: ({ message, complexity }) => (
        complexity.score > 0.8 &&
        !knowledgeBase.hasAnswer(message)
      ),
      action: {
        notify: 'slack:engineering',
        priority: 'medium',
        draftResponse: 'best-effort',
        flagForReview: true
      }
    },

    {
      name: 'competitor-mentioned',
      detect: ({ message }) => detectCompetitors(message).length > 0,
      action: {
        notify: 'slack:founders',
        priority: 'medium',
        competitors: detectCompetitors(message),
        draftResponse: 'respectful-differentiation',
        requireApproval: true
      }
    },

    {
      name: 'custom-integration-request',
      detect: ({ message }) => (
        message.includes('custom integration') ||
        message.includes('API access') ||
        message.includes('white label')
      ),
      action: {
        notify: 'slack:engineering',
        priority: 'low',
        aiResponse: 'gather-requirements',
        escalateAfterRequirements: true
      }
    }
  ],

  // Human review (optional, not blocking)
  review: [
    {
      name: 'first-outreach-to-target-account',
      detect: ({ lead }) => lead.tags.includes('target-account'),
      action: {
        requireApproval: true,
        draftEmail: true,
        allowOverride: true,
        timeout: '4h' // Auto-send if not reviewed in 4 hours
      }
    },

    {
      name: 'high-profile-prospect',
      detect: ({ lead }) => (
        lead.linkedinFollowers > 10000 ||
        lead.companyProfile === 'well-known'
      ),
      action: {
        requireApproval: false,
        notifyForReview: true,
        draftEmail: true
      }
    }
  ]
})

## The Collaboration Model

Humans and AI work together:

**AI's Role:**
- Handle all routine interactions
- Research and enrich leads
- Draft all communications
- Identify escalation triggers
- Provide context when escalating
- Execute after human decisions
- Learn from human edits

**Human's Role:**
- Handle strategic conversations
- Make final decisions on pricing/terms
- Build executive relationships
- Resolve complex objections
- Approve high-stakes communications
- Provide creative direction
- Review and improve AI outputs

### 3. Clear Escalation Criteria

Document exactly when to escalate:

```markdown
## Escalation Criteria

### Always Escalate:
- Deal size > $50K ARR
- Executive (CEO, CTO, Founder) reaching out
- Legal, security, compliance questions
- Partnership/acquisition inquiries
- Angry/frustrated customers
- Custom enterprise requirements

### Escalate If:
- Competitor mentioned AND high-value lead
- Complex technical question AND no KB answer
- Multiple objections in single message
- Request for custom pricing
- Mention of specific deadline/urgency

### Never Escalate:
- Routine follow-ups
- Simple product questions
- Meeting scheduling
- Content delivery
- Standard objections (price, timing, etc.)
```
