## The Four Function Primitives

Sales-builder's architecture is built on four types of functions that mirror how modern growth teams actually work:

### 1. Code Functions: The Analytics Brain

Code functions handle tracking, analytics, and rule-based automation:

```typescript
import { defineCodeFunction } from '@sales-builder/functions'

export const leadScoring = defineCodeFunction({
  name: 'calculate-lead-score',
  trigger: 'lead.created',
  execute: async ({ lead, db }) => {
    const score =
      (lead.companySize > 50 ? 20 : 0) +
      (lead.title.includes('VP') ? 25 : 0) +
      (lead.industry === 'SaaS' ? 15 : 0) +
      (await db.engagement.count(lead.email) * 5)

    return { score, priority: score > 60 ? 'high' : 'medium' }
  }
})
```

### 2. Generative Functions: The Content Engine

Generative functions create personalized content at scale:

```typescript
import { defineGenerativeFunction } from '@sales-builder/functions'

export const personalizedOutreach = defineGenerativeFunction({
  name: 'craft-outbound-email',
  model: 'claude-opus-4.5',
  prompt: `Write a personalized outreach email to {{lead.name}} at {{lead.company}}.

  Context:
  - Their role: {{lead.title}}
  - Recent company news: {{lead.recentNews}}
  - Our value prop: Autonomous sales agents that replace entire growth teams

  Tone: Professional, concise, value-focused. Max 100 words.`,

  variables: ['lead.name', 'lead.company', 'lead.title', 'lead.recentNews']
})
```

### 3. Agentic Functions: The Autonomous Workforce

Agentic functions make decisions and take actions independently:

```typescript
import { defineAgenticFunction } from '@sales-builder/functions'

export const autonomousResponder = defineAgenticFunction({
  name: 'handle-inbound-questions',
  trigger: 'email.received',
  capabilities: [
    'answer-product-questions',
    'schedule-demos',
    'provide-pricing',
    'escalate-to-human'
  ],

  decision: async ({ email, context }) => {
    // AI decides whether to respond, escalate, or ignore
    if (email.sentiment === 'frustrated') {
      return { action: 'escalate', priority: 'high' }
    }

    if (email.hasQuestion) {
      return { action: 'respond', generateAnswer: true }
    }

    return { action: 'log', addToNurture: true }
  }
})
```

### 4. Human Functions: Strategic Oversight

Human functions handle high-value decisions and relationship building:

```typescript
import { defineHumanFunction } from '@sales-builder/functions'

export const finalizeEnterpriseDeal = defineHumanFunction({
  name: 'close-enterprise-deal',
  trigger: 'deal.stage.negotiation',

  notify: {
    channel: 'slack',
    include: ['deal-summary', 'conversation-history', 'pricing-analysis']
  },

  timeout: '24h', // escalate if not completed

  onComplete: async ({ decision, deal }) => {
    if (decision === 'approve') {
      await sendContract(deal)
      await scheduleOnboarding(deal)
    }
  }
})
```
